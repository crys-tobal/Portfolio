About Me

Hello, my name is Cristobal Caayao, but you can call me Cris!

I'm a UX / UI Designer passionate about designing intuitive experiences and coding responsive websites. My work revolves around solving problems both for clients and for the end users by using an empathetic approach to design, research and testing.

I had been studying architecture for years because I was drawn to creating solutions that were both beautiful and functional. However, after working in the industry I realised it just wasn't for me.

I've only been coding for a year, but I think I've found my calling as a web designer and developer. At least, it's a lot simpler than building a house!
